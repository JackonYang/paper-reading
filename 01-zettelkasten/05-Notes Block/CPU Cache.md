## 高速缓存

m - 总地址长度，分 s 个 cache set(缓存组)，每组 e 个 cache line. 每个 cache line 是 b 个字节的 block。另外有 1 个 valid bit，t 个标记位

t 表示，组内的第几个 cache line。当 valid bit 有效时，生效。
